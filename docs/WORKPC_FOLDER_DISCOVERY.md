# Werk-pc Outlook Folder Discovery

Deze stap is alleen bedoeld om te zien welke Outlook stores en mappen Python kan zien.

Toegestaan:
- Outlook folders oplijsten;
- mapnaam, pad, totaal aantal items en ongelezen aantal lezen;
- lokaal rapport maken in `reports/`.

Niet toegestaan en niet aanwezig in deze tool:
- mails openen/exporteren;
- mails markeren als gelezen;
- mails verplaatsen of verwijderen;
- antwoorden, doorsturen of verzenden;
- categorieen of flags wijzigen;
- mailboxmappen aanpassen.

## Start op de werk-pc

```bat
run_workpc_folder_discovery.bat
```

Daarna ontstaat een rapport in:

```text
reports\outlook_folder_discovery_YYYYMMDD_HHMMSS.md
```

Bezorg dat rapport terug voor analyse.
